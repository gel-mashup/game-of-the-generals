    **server/src/socket/handlers/roomHandler.ts:**
    - Import customAlphabet from nanoid
    - Generate 6-char room codes using: customAlphabet('ABCDEFGHJKLMNPQRSTUVWXYZabcdefghjkmnpqrstuvwxyz23456789', 6)
    - Store rooms in Map<string, Room>
    - Handle 'create-room' event:
      - Generate unique roomId
      - Create room with status 'waiting', empty board (8x9), isBotGame flag
      - socket.join(roomId)
      - emit 'room:created' with { roomId, playerId: socket.id, playerSide: 'red', isBotGame }
    - Handle 'join-room' event:
      - Validate roomId exists
      - If room full (>2 players or bot game), emit 'error' with "Room is full"
      - socket.join(roomId)
      - emit 'room:joined' with { roomId, playerId: socket.id, playerSide: 'blue' }
      - socket.to(roomId).emit 'player:joined' with { player }
    - Handle 'leave-room' event:
      - socket.leave(roomId)
      - Remove player from room.players
      - If room empty, delete room from Map
      - socket.to(roomId).emit 'player:left' with { playerId, reason }
