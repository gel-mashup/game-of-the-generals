## Milestone Summary

**Key Decisions:**
- Separate Express+Socket.io server (port 3001) from Next.js (port 3000)
- React Context pattern for Socket.io (SocketProvider)
- Two Zustand stores: gameStore + roomStore
- CSS Grid 9x8 board with alternating cell colors
- Spy beats all officers; only Private can beat Spy
- Battle resolution priority: flag → spy/private → equal rank → higher rank
- Minimax depth 3 with iterative deepening and 3s time limit
- Multi-stage Docker builds with non-root users

**Bugs Fixed During Development:**
- Battle tie piece survival (attacker re-placed after elimination)
- Cannot attack adjacent enemy pieces (click handler clearing selection)
- Bot rematch stuck (client round-trip deploying wrong side)
- Bot stuck on turn (null findBestMove not triggering game-over)
- Rematch button not emitting socket event (WinModal onClick bug)
- Flag movement rule change (flags can now move)
- Game not starting after room join (sync-game-state mechanism)

**Technical Debt:**
- DEP-02 (zone enforcement) and GAME-04/05/08/09/10 (game mechanics) implemented but traceability checkboxes were never updated during rapid YOLO development
